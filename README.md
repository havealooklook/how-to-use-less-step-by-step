LESS学习
============
CSS预处理让 CSS 更易维护、方便制作主题、扩充，并且可以大幅度提高工作效率。LESS预处理语言易于学习,可以运行在 Node、浏览器和 Rhino 平台上。网上有很多第三方工具帮助你编译 Less 源码,此项目使用gulp-less。<br>

##项目结构
* /less/<br>
	less文件，并附有注释
* /css/<br>
	编译完成的css文件
* /*.html
	html文件，可查看效果
